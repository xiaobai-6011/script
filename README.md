# 脚本合集

## ocserv VPN 安装脚本

### 功能列表

| 菜单 | 功能 |
|------|------|
| 1 | 安装 VPN |
| 2 | 配置 VPN |
| 3 | 启动 VPN |
| 4 | 停止 VPN |
| 5 | 重启 VPN |
| 6 | 查看状态 |
| 7 | 添加用户 |
| 8 | 删除用户 |
| 9 | 修改欢迎信息 |
| 10 | 查看在线用户 |
| 11 | 流量统计 |
| 12 | 修改端口 |
| 13 | 重新生成证书 |
| 14 | 查看日志 |
| 15 | 卸载 VPN |

### 使用方法

```bash
# 安装
sudo ./ocserv.sh install

# 添加用户
sudo ./ocserv.sh add username password

# 查看状态
sudo ./ocserv.sh status
```

### 配置说明

- 网段：172.16.0.0/22 (1017个IP)
- 端口：443
- 路由模式：内网分流
